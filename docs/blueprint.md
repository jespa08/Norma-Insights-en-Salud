# **App Name**: Norma Insights

## Core Features:

- Query Interpretation: Intelligent query understanding to interpret complex queries related to the regulatory, operational, financial, administrative, human, and technological aspects of the healthcare system in Colombia and Mexico.
- Validity and Context Validation: AI tool to validate the relevance and context of information extracted from regulatory documents.
- Report Generation: Generate comprehensive consulting-style reports with data support and citations. Report format should emulate those produced by large firms and research centers. Generative AI will produce reasoning and formatting for the consulting-style reports.
- Query Submission Interface: Present a user-friendly interface for submitting queries and viewing report generation progress.
- PDF Export: Enable users to download the final output as a PDF document.
- Usage Limitation: Limit usage to two consultations per user per day.

## Style Guidelines:

- Primary color: Deep blue (#1E3A8A) to convey trust, stability, and intelligence, reflecting the authoritative nature of the app's insights. The deep blue will also make the consulting papers look like corporate-approved assets.
- Background color: Light gray (#F0F0F5), nearly white, provides a clean, professional backdrop that emphasizes the content.
- Accent color: Soft teal (#74C69D) to highlight key insights and interactive elements, offering a modern and approachable contrast. This teal complements the blue in a balanced way.
- Body: 'Literata', serif, for readability of the consulting reports. Headlines: 'Inter', sans-serif, for clear titles.
- Use professional, minimalist icons to represent different sections and functionalities, aiding navigation and enhancing usability.
- Employ a clean and structured layout to facilitate easy navigation and readability, ensuring key information is easily accessible.
- Subtle loading animations during report generation to keep users engaged without distraction.
